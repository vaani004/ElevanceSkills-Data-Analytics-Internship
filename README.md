# ElevanceSkills Data Analytics Internship

## Project Overview

This repository contains the work completed during the ElevanceSkills Data Analytics Internship. The project focuses on data visualization and analytics using Tableau dashboards built on a common job market dataset.

## Tools Used

- Tableau Public
- Microsoft Excel / CSV Dataset
- GitHub

## Dataset

The dashboards were created using a job descriptions dataset containing information such as:

- Company Name
- Company Size
- Job Title
- Salary
- Qualification
- Work Type
- Country
- Job Portal
- Contact Person

## Tasks Completed

### Task 1
Analysis dashboard based on job preferences and company-related metrics.

### Task 2
Company Size Comparison Dashboard.

### Task 3
Work Type Salary Distribution Analysis.

### Task 4
India vs Germany Job Postings Comparison Dashboard.

### Task 5
Top 10 Companies Hiring Data Scientists (Treemap Visualization).

### Task 6
Qualification Drilldown Map using Geographic Visualization and Interactive Filters.
